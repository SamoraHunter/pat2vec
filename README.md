# pat2vec
Transform's single patients data into single row vector.

Transform batch patient data into data matrix D ready for machine learning (binary classification). 

## Requirements:

CogStack (cogstack_v8_lite)
elasticsearch
medCat

csv
multiprocessing
os
re
sys
warnings
numpy
pandas
pickle
scipy
COGStats
json
random
subprocess
time
traceback
datetime
functools
io
pathlib
tqdm
colorama


### Features:

- Single patient
- Batch patient
- Cohort search and creation
- Automated random controls
- Modular feature space selection
